## Diretrizes de Imagem

- Use imagens de alta qualidade que representem telemedicina e cuidados de saúde
- Otimize os tamanhos de arquivo para web (comprima as imagens antes de fazer upload)
- Use PNG para imagens com transparência, JPG para fotografias
- Certifique-se de que as imagens são profissionais e alinhadas com a identidade da marca
- Paleta de cores deve harmonizar com:
  - Primária: #1A4B84 (azul escuro)
  - Secundária: #26A69A (verde-azulado)
  - CTA: #00BCD4 (ciano)
  - Fundo: #F4F7F9 (cinza claro)

## Notas Importantes

- **O layout está preparado para funcionar mesmo sem as imagens**
- Quando as imagens não são encontradas, placeholders coloridos são exibidos automaticamente
- As imagens devem ser adicionadas posteriormente sem quebrar o layout existente
- Todos os componentes têm tratamento de erro para imagens ausentes
- Os placeholders são baseados nas cores da paleta oficial da marca
